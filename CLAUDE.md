# CLAUDE.md

Этот файл содержит инструкции для Claude Code (claude.ai/code) при работе с данным репозиторием.

## О проекте

Корпоративный лендинг для ООО Энергомонтаж-55 — компании по электромонтажным работам из Санкт-Петербурга. Написан на чистом HTML5, CSS3 и JavaScript — без сборщиков и пакетных менеджеров.

## Локальный запуск

```bash
php -S localhost:8000
```

PHP нужен только для обработки контактной формы (`main.php`). Остальное работает на любом статическом сервере.

## Архитектура

**Четыре основных файла:**
- `index.html` — одностраничная разметка со всеми секциями
- `index.css` — все стили (чистый CSS, без препроцессора)
- `index.js` — вся клиентская логика
- `main.php` — обработчик формы обратной связи (отправка через PHP `mail()`)

**Секции страницы (по порядку):** шапка/навигация → hero `.home` → о компании `.features` → услуги `.fields` → карусель партнёров `.partners` → карта проектов `.projects` → форма `.question` → подвал

## Ключевые части index.js

- **Динамический расчёт возраста** — количество лет компании считается автоматически от 2006 года
- **Яндекс Карты** — два экземпляра карты: `initProjects()` (19 кластеризованных меток проектов) и `initAddress()` (офис в подвале); координаты проектов захардкожены массивом (строки 16–36)
- **Owl Carousel** — карусель логотипов партнёров, адаптивная (2 элемента на мобильных / 5 на десктопе)
- **Маска телефона** — библиотека IMask, формат `+7(___) ___-__-__`
- **Отправка формы** — XHR POST на `/main.php`, без перезагрузки страницы
- **Мобильная навигация** — гамбургер-меню через Drawer.js

## Сторонние зависимости

Все зависимости — либо локальные файлы, либо CDN-ссылки в `index.html`. Файла `package.json` нет.

Локальные (в репозитории):
- `jquery-3.6.0.min.js`
- `owlcarousel/` — Owl Carousel 2
- `wow/wow.min.js` — анимации при скролле

CDN:
- Яндекс Карты API (ключ в `index.js`)
- Animate.css
- Drawer.js + iScroll 5.2.0
- Font Awesome
- Google Fonts (Montserrat)
- IMask (форматирование телефона)
